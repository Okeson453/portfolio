## 📖 How to Use

### In IDE
1. Open the `docs/` folder in your editor
2. Start with `docs/INDEX.md`
3. Click links to navigate between documents

### In Terminal
```bash
# List all documentation
ls docs/**/*.md

# Search for specific content
grep -r "keyword" docs/

# Count total files
find docs -name "*.md" | wc -l

# View specific category
ls docs/performance/
ls docs/deployment/
```

### In Browser (if deployed)
- Mount docs folder on your documentation site
- Use INDEX.md as landing page
- Serve with Markdown renderer
