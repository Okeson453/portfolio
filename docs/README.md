# Secure Stack Portfolio 🛡️

A modern, interactive cybersecurity portfolio built with Next.js 14, TypeScript, and Tailwind CSS. Features a cyberpunk aesthetic with real-time security visualizations.

## ✨ Features

### 🎨 **Modern UI/UX**

- Cyberpunk theme with neon accents
- Smooth animations & transitions using Framer Motion
- Dark/light mode support
- Responsive design for all devices
- Interactive 3D elements with Three.js

### 🔒 **Security Focus**

- Security headers & CSP configuration
- Secure contact form with EmailJS
- XSS protection
- Performance optimizations
- Accessibility (WCAG 2.1 AA compliant)

### 📊 **Interactive Elements**

- Real-time vulnerability scanner visualization
- Interactive project showcases
- Animated skill graphs
- Terminal-style command interface
- Network activity simulation

### 🛠️ **Developer Experience**

- TypeScript for type safety
- ESLint & Prettier configuration
- Component-based architecture
- Reusable UI components
- Comprehensive testing setup
